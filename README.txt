How to use this portfolio website
=================================

1. Open index.html in a browser to preview the site.
2. Edit these placeholders before publishing:
   - "Your Name"
   - "your.email@example.com"
   - LinkedIn URL
   - Avatar initials (currently YN)
3. If you want to deploy quickly, upload the folder contents to Netlify, Vercel, GitHub Pages, or any static hosting.
4. Main files:
   - index.html  -> content/structure
   - styles.css  -> visual design/responsive layout
   - script.js   -> simple reveal animation
